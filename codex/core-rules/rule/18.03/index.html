18.03 Disembarking
x

In the active player’s Movement phase, each friendly unit embarked within a TRANSPORT model can disembark from it by making a disembark move (18.04).
If a TRANSPORT model is destroyed, before removing it from the battlefield, the active player must make an emergency disembark move (18.05) with each unit embarked within it.

FAQ

If a unit makes any disembark move in a turn (i.e. Disembark move, Emergency Disembark move etc.) has it disembarked that turn?

Yes.

FAQ

Aerialists ability allows you to get into a transport even though they disembarked a transport this turn?

Yes, as the unit embarking is not making a normal, advance or fall-back move as part of this embarkation.
