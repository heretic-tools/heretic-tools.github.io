08. Command Phase
x

In the Command phase, both players gain Core CP, then you check the combat readiness of your forces and use any strategic-level abilities you may have.

The Command phase is resolved by following these steps:

1. START OF COMMAND PHASE
2. GAIN CORE CP
3. BATTLE-SHOCK
4. COMMAND ABILITIES
5. END OF COMMAND PHASE
