Leader

Core

Before the battle, in the Muster Armies step, for each leader and support unit in your army, you can select one friendly bodyguard unit that unit can lead. That unit will then lead that bodyguard unit for the battle and form an attached unit with it.

Unless otherwise stated, each bodyguard unit can only have one leader unit and one support unit attached to it.

Deep Strike

Core

Each time this unit makes an ingress move (20.04), if every model in this unit has this ability, it can be set up anywhere on the battlefield that is more than 8" horizontally from all enemy units, even if that is within your opponent’s deployment zone.

There are many ways by which to deploy troops to the field of battle including tunnels, teleportation devices and other esoteric means of transportation.

FAQ

When an Ogryn Bodyguard joins a Militarum Tempestus Command Squad, can the unit set up using the Deep Strike ability?

No, as the Ogryn Bodyguard model does not have the Deep Strike ability.

Lore

Led by driven officers known as Tempestor Primes, the Command Squads of the Militarum Tempestus comprise veteran Tempestus Scions. These specialists guard and support the Tempestor Prime while the officer marshals their elite soldiers into the perfect positions for the mission at hand.
